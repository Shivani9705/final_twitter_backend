# final_twitter_backend
Twitter Clone (MERN Stack)

A full-stack social media web application inspired by Twitter. Users can create accounts, post tweets, follow/unfollow others, and view real-time feeds.

Live Demo

🔗 Frontend: https://incandescent-selkie-a51894.netlify.app/

🔗 Backend API: https://final-twitter-backend-3g44.onrender.com/

🛠️ Tech Stack

Frontend: React.js, Context API, CSS 

Backend: Node.js, Express.js

Database: MongoDB Atlas

Authentication: JSON Web Token (JWT)

Hosting: Netlify (Frontend), Render (Backend)
